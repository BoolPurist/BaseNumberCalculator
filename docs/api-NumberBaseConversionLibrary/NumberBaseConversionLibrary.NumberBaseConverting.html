Class NumberBaseConverting

Class provides a way of converting a text number with a certain base into either a integer or a string as a number with a different base.

Examples

Converting text number "A4" with base 16 into the number "10010100" with base "2". That example shows how to do it.

string sourceNumber = "A4";
int sourceInteger = NumberBaseConverting.NumberValueFromTextNumber(sourceNumber, 16);
// sourceInteger = 164, with the method NumberValueFromTextNumber(String, Int32)
string targetNumber = NumberBaseConverting.TextNumberFromNumberValue(sourceInteger, 2 );
// tragetNumber = "10010100", with the method TextNumberFromNumberValue(Int32, Int32)
// targetNumber is the result.

Returns
Type Description
System.Int32

Numeric value that comes from the conversion out of the text number textNumber. Returns -1 if the text number has at least one char with values above the sourceBase. Example: With "A4" for textNumber and 10 for sourceBase returns -1, because 'A' only exit in a number systems with a base of above 10.
Exceptions

System.ArgumentOutOfRangeException

Occurs if parameter sourceBase is zero or smaller. Occurs if the parameter is greater than the value of MaxBase.
System.ArgumentException

Occurs if the textNumber is an empty string or has no char which is from 0-9, a-z or A-Z.
