Can convert a number with a base into the number with a different number.
Example: Number "A4" with base 16 can converted to the number with the base 10. In this case it is "164".

See PictureManuel.png what is what. 

1. Enter a number in the number input field. Number must be a char a digit or letter from a to z or A to Z.
2. Enter a positive number as the source base of the given number in the source base input field.
3. Enter a positive number as the target base of the resulting number. 
4. Click the button that says  "Convert".
5. Now you see the result in green below the convert button.
